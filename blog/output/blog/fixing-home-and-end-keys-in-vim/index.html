Fixing home and end keys in Vim

Ariel Gerardo Ríos

2016-12-29 01:03

Comments

Código fuente

Sitio original

Adding this to ~/.vimrc file fixes incorrect HOME & END key behavior:

" Home key
imap <esc>OH <esc>0i
cmap <esc>OH <home>
nmap <esc>OH 0

" End key
nmap <esc>OF $
imap <esc>OF <esc>$a
cmap <esc>OF <end>
