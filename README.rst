ariel17.com.ar

The ariel17.com.ar_ web site content. This is a personal page where I talk
about myself to fatigue. There is a humble blog too :)

Building image
--------------

```bash
$ docker build -t ariel17/www .
$ docker run -p 8080:80 ariel17/www
``
